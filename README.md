## Inventory Management System

An inventory management system developed to manage products, track stock levels, and record sales transactions efficiently.

### Project Description
This system is designed to help organize inventory data by maintaining product records, monitoring available stock, and recording sales activities. It aims to improve accuracy and efficiency in inventory handling.

### Tools Used
- SQL
- Database Design

### Features
- Product management
- Stock level tracking
- Sales record management
- Organized inventory data

### Purpose
This project was developed as an academic and practice project to strengthen database design and data management skills.
